# Valo Black Hat Guided Demo

The Community Edition includes a polished guided demonstration at:

```text
http://localhost:8080/demo
```

## Included experience

- One-click **Run complete demo** workflow
- Four preloaded prompt and policy scenarios
- Live calls to the Valo enforcement simulation API
- Allow, review, and block decision storytelling
- LLMShadow, SaaSShadow, and AgentShadow sample findings
- Executive KPI and risk-distribution views
- Print-ready executive report export using **Save as PDF**

## Recommended presenter flow

1. Open **Guided Demo** from the left navigation.
2. Explain the portfolio view: Valo Core, LLMShadow, SaaSShadow, and AgentShadow.
3. Select **Run complete demo**.
4. Walk through the benign, PII, credential, and prompt-injection scenarios.
5. Review the unified exposure cards.
6. Select **Export executive PDF** and choose **Save as PDF** in the browser print dialog.

The portfolio findings are clearly labeled as illustrative demonstration data. Policy simulation results are produced through the live local Valo API.
